# Správa kuchyně a receptů
**Autor:** Jan Novák  
**Datum:** Duben 2024

## Web
Aplikace poskytuje interaktivní rozhraní pro prohlížení a filtrování receptů. Využívá moderní CSS Grid pro responsivní layout karet.


## Databáze
Relační schéma je navrženo v 3. normální formě s využitím vazební tabulky pro vztah M:N mezi recepty a ingrediencemi.


## Program
Obslužný skript v Pythonu zajišťuje zpracování datových exportů, agragaci duplicitních ID a přepočet surovin dle počtu strávníků.
